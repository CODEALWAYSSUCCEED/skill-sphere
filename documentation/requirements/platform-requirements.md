## 1. User Accounts
- Users must be able to create an account.
- Users must be able to log in and log out.
- User authentication must be handled using Firebase.
- Each user must have a role (user or mentor).

## 2. Skill Browsing
- Skills must be stored in Firestore.
- Users must be able to view skill cards.
- Each skill must display SDG alignment.

## 3. Reflection System
- Users must be able to write reflections.
- Reflections must be saved to Firestore.
- Reflections must link to the correct user.

## 4. Learning Groups
- Users must be able to join groups.
- Mentors must be able to create groups.
- Group membership must be stored in Firestore.

## 5. Progress Tracking
- Skill progress must update automatically.
- Progress must be displayed as a percentage.

## 6. Non-Technical Requirements
- Clean UI
- Responsive layout
- Clear navigation
- Error handling
